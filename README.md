# Computer Graphics Lab

A collection of Computer Graphics laboratory programs implemented in C using the BGI graphics library (`graphics.h`).

## Lab Experiments

| Lab | Topic |
|-----|-------|
| Lab 01 | DDA Line Drawing Algorithm |
| Lab 02 | Bresenham's Line Drawing Algorithm |
| Lab 03 | Bresenham's Circle Drawing Algorithm |
| Lab 04 | Midpoint Ellipse Drawing Algorithm |
| Lab 05 | 2D Transformations (Translation, Rotation, Scaling, Reflection) |
| Lab 06 | Cohen-Sutherland Line Clipping Algorithm |
| Lab 07 | Sutherland-Hodgman Polygon Clipping Algorithm |
| Lab 08 | Bezier Curve Drawing |
| Lab 09 | Flood Fill Algorithm |
| Lab 10 | Boundary Fill Algorithm |

## How to Compile and Run

These programs use the BGI (`graphics.h`) library available in **Turbo C / Dev-C++ with WinBGIm**.

### Using Dev-C++ with WinBGIm
1. Download and install [Dev-C++](https://sourceforge.net/projects/orwelldevcpp/)
2. Install the WinBGIm package (graphics.h for Windows)
3. Open the `.c` file, compile and run

### Using Turbo C
1. Open Turbo C
2. Load the `.c` file
3. Compile with `Alt+F9` and Run with `Ctrl+F9`
